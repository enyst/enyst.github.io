Methodology notes

Counts include GitHub PR review events with state APPROVED, CHANGES_REQUESTED, or COMMENTED. Self-review events are excluded: if the reviewer is also the PR author, that event is not counted in the tables or charts. PR author buckets follow the latest explicit criteria used in the source JSON.
