Repositories

    Review event rule

    Counts include GitHub PR review events with state APPROVED, CHANGES_REQUESTED, or COMMENTED. Events where the reviewer is also the PR author are excluded.

    Methodology notes

    Reviewer cohort is the union of employees and maintainers from the community PR dashboard criteria plus the explicit overrides in the reproducibility script. The charts and tables are generated from the aggregate JSON file below.

    Employee / maintainer source

    Employees are based on visible OpenHands organization membership plus script overrides. Maintainers are based on the dashboard maintainer allowlist. If someone is both, the role label displays maintainer.
