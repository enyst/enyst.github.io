# Positioning

## What this is
A personal technical site for Engel Nyst — software developer, open source maintainer, builder of AI agent systems.

## What it's not
- Not a company landing page
- Not a portfolio with client testimonials
- Not a blog with engagement metrics
- Not a resume site (the resume exists, but the site is the work itself)

## Market position
This is an architect's notebook made public. Among developer personal sites, it sits in the "shows real work in progress" category — not polished case studies, not just a list of repos, but actual technical artifacts: architecture diagrams, investigation reports, data dashboards, working prototypes.

## Core promise
You'll see how things actually work. Not how they're marketed.

## What makes it different
Most developer sites are either bare-bones (name + GitHub link) or overproduced (stock photos, buzzword copy). This one publishes the actual technical work — diagrams of real systems, forensic contribution reports, live architecture documents. The site itself is built with the same tools it documents.
