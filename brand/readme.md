# Engel Nyst — Brand Atomic Kit

This folder contains the brand system for enyst.github.io in two formats:

- `/human/` — prose guidelines for humans
- `/agent/` — structured files that AI agents can read and build from

## How to use

Drop the `/agent/` folder into any AI coding tool (Cursor, Claude, Codex, OpenHands) and prompt:

> "Generate an HTML page using this brand system."

The output should have the right colors, fonts, tone, and feel without any additional instructions.

## The aesthetic

**A builder's journal by lamplight.** Dark workshop, warm parchment ink, gold accents catching light. Technical but never cold. Things under construction, shared openly. Diagrams, investigations, architecture notes — the work itself is the portfolio.

## The voice

Direct. Technical but readable. Occasionally dry. Never corporate, never performative. The kind of writing where every sentence earned its place.
